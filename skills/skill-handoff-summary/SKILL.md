# Skill Handoff Summary

## Overview

Capture a clear pause or transfer summary for a coding task that will be resumed later by the same person or another agent.
Use this skill for handoff quality, not for rebuilding stale context or deciding phase gates.

## Use This Skill When

- pausing work and needing a clean summary for later resumption
- transferring a task to another agent with open questions and blockers
- capturing current status, risks, and next steps in one place
- reducing restart cost after a stop point in a long thread
